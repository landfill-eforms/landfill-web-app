About Landfill e-Forms

Introduction

The Landfill e-Forms project was devolped as part of the senior design program at California State University, Los Angeles.

The purpose of this project is to convert the City of Los Angeles, Department of Sanitation's methane emission auditing system from paper to fully electronic in order to improve efficiency, reduce human error, and speed up report production. The project consists of the development two applications: a mobile application for on-field use and a web application for desktop use. The mobile application is developed for the Android Operating System and will allow the user to enter and search data on the mobile device. The web application was developed with using Angular 4 on the front-end, Java/Spring on the back-end, and Microsoft's SQL Server for the database. The web application serves as the main source for data validation, data search, and generating reports. Data can be synced between the web application and the mobile devices via hardwired sync.
